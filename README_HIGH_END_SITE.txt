RATIOS High-End Nonprofit Website

HOW TO INSTALL
1. Extract this ZIP.
2. Copy EVERYTHING inside this folder into your local GitHub repository folder.
3. Choose Copy and Replace when Windows asks.
4. In GitHub Desktop, you should see changed files.
5. Commit with: High-end RATIOS nonprofit website redesign
6. Click Push origin.
7. Cloudflare Pages will redeploy automatically.

IMPORTANT FEATURES INCLUDED
- Smooth reveal animations
- Animated counters
- Donation progress section
- Stripe/Zeffy placeholder donation links
- Interactive safety kit display
- Service area map graphic
- Founder story page
- News & Events page
- Volunteer and resource request forms
- Contact forms using FormSubmit
- Sponsor logo placeholder strip
- Polished mobile navigation

FORMS
Forms currently post to FormSubmit using ratiossaveslives@gmail.com. The first submission will send an activation email. You may replace the email in the HTML files with your preferred RATIOS email.

DONATIONS
Replace the placeholder Stripe test links in donate.html with your real Stripe Payment Link, Zeffy donation link, or PayPal donation link.

IMAGES
Some images are illustrative AI-generated awareness images. Avoid presenting them as actual clients or testimonials unless you have consent from real people.
