HOW TO INSTALL

1. Download and extract the ZIP.
2. Open the extracted folder.
3. Select:
   - ready-set-save.html
   - assets
4. Copy both items.
5. Paste them directly into:
   C:\Users\Desirae\Documents\GitHub\ratios-website
6. When Windows asks, choose:
   Replace the files in the destination
7. Open GitHub Desktop.
8. Confirm the updated HTML and image files appear under Changes.
9. Commit with:
   Update Ready Set Save design
10. Click Push origin.
11. Check the BLUE Cloudflare Pages project for a successful deployment.
12. Open:
   https://ratiossaveslives.org/ready-set-save.html
