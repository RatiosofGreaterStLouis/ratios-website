RATIOS Modern Website - Free Hosting Instructions

WHAT IS INCLUDED
- Multi-page website: Home, About, Programs, Safety Kits, Ready Set Save, Donate, Volunteer, Resource Request, Gallery, Contact
- Custom CSS and JavaScript
- RATIOS shirt and community images
- Mailto-based forms that open the visitor's email app

IMPORTANT CUSTOMIZATION
1. Replace hello@ratiossaveslives.org with your real nonprofit email if different.
2. Replace donation placeholder buttons with your donation platform link when ready.
   Free/low-cost options to look at: Zeffy, Givebutter, PayPal Giving Fund, Donorbox, Stripe.
3. Forms currently use email links. For a real web form, connect Google Forms, Jotform, Tally, Formspree, Netlify Forms, or your website host's forms.
4. AI imagery disclaimer is already included in the footer and gallery.

EASIEST FREE HOSTING OPTION: NETLIFY DROP
1. Go to app.netlify.com/drop
2. Drag the entire ratios_modern_site folder onto the page.
3. Netlify will publish the website and give you a temporary free URL.
4. To connect your GoDaddy domain, go to Domain Management in Netlify and follow the DNS instructions.

FREE OPTION: CLOUDFARE PAGES
1. Create a free Cloudflare account.
2. Go to Workers & Pages > Create > Pages.
3. Upload this folder, or connect a GitHub repo.
4. Connect ratiossaveslives.org and update GoDaddy nameservers/DNS as Cloudflare instructs.

FREE OPTION: GITHUB PAGES
1. Create a GitHub account.
2. Create a new repository named ratiossaveslives.
3. Upload all files and folders from this website folder.
4. Go to Settings > Pages.
5. Choose Deploy from branch, main branch, root folder.
6. Connect your GoDaddy domain in Pages settings.

GODADDY DOMAIN NOTE
Owning the domain does not automatically include website hosting. You can keep the domain at GoDaddy and host the actual website for free somewhere else.
