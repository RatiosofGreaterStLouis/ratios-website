INSTALLATION
1. Extract this ZIP.
2. Copy ready-set-save.html and the assets folder.
3. Paste both directly into:
   Documents > GitHub > ratios-website
4. Choose “Replace the files in the destination.”
5. Open GitHub Desktop.
6. Confirm the HTML file and four JPG files appear under Changes.
7. Commit with:
   Separate Ready Set Save class photos
8. Click Push origin.
9. After Cloudflare Pages deploys, visit:
   https://ratiossaveslives.org/ready-set-save.html

Do not paste the outer update folder into the repository. Copy its contents directly.
